# Projeto Frontend - Geração Tech 3.0

Projeto frontend para o curso **Desenvolvedor Web Full Stack** do **Geração Tech 3.0**. O objetivo desse projeto é a aplicação dos conhecimentos aprendidos no curso, resultando na criação de um site baseado em **React** e **Bootstrap**. Esse projeto foi feito por **Pedro Lucas Ferreira Santana** totalmente a mão, sem uso de IA.

#### Foram utilizadas as seguintes tecnologias:
[React](https://github.com/facebook/react)<br />
[Bootstrap](https://github.com/twbs/bootstrap)<br />
[React-Bootstrap](https://github.com/react-bootstrap/react-bootstrap)<br />
[React-Router](https://github.com/remix-run/react-router)<br />
[Sass](https://github.com/sass/sass)<br />
[Vite](https://github.com/vitejs/vite)<br />
[Vite Deploy](https://github.com/ErickKS/vite-deploy)

#### Como testar o projeto:
1. Clone o repositório:
   ```bash
   git clone https://github.com/pedroluc97/gtech3-projeto-frontend.git
   ```
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Rode o servidor local:
   ```bash
   npm run dev
   ```  
4. Acesse o site em seu navegador de preferência:
   ```
   http://localhost:5173/
   ```
5. Ou, alternativamente, accesse o site via o deploy:<br />
   [https://pedroluc97.github.io/gtech3-projeto-frontend/](https://pedroluc97.github.io/gtech3-projeto-frontend/)
